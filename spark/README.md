# Spark Runtime

Run locally:

```bash
bash ./spark/start.sh
```

Open `http://127.0.0.1:3100`.

Verify:

```bash
node --test ./test/spark-runtime.test.js ./test/spark-server.test.js ./test/spark-security.test.js
npm run verify
```

Spark stores runtime state under `.cyvx/`, creates portable World assets, records evidence, captures leads, queues follow-up work, and preserves owner control over approval, interruption, configuration, and export.
