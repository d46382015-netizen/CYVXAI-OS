# CYVX Connector Specs

## Purpose

CYVX now has connector specifications for the three highest-leverage support layers:

- Agent orchestration
- Workflow execution
- SBOM/security proof

## Connectors

### Agent Framework

Role: agent orchestration  
Value: converts CYVX missions into agent plans, assignments, execution traces, and reviews.

### Activepieces

Role: workflow execution  
Value: turns approved CYVX missions into executable automations.

### Syft

Role: security/SBOM proof  
Value: produces dependency inventory, SBOM evidence, and repository risk posture.
